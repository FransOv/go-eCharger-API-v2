# Modbus

Die go-eCharger bieten ein Modbus TCP Interface an, mit dem einige Leistungs- und Energiewerte abgefragt werden können. Dieses Interface muss erst über die App oder [http API](http-de.md) aktiviert werden. Daten gelesen werden können über die [Modbus Funktion 3](https://www.simplymodbus.ca/FC03.htm) und die [Modbus Funktion 4](https://www.simplymodbus.ca/FC04.htm). Daten geschrieben werden kann über die [Modbus Funktion 15](https://www.simplymodbus.ca/FC15.htm) und die [Modbus Funktion 16](https://www.simplymodbus.ca/FC16.htm).
